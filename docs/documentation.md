# Документация Geomeric Lib

## Общее описание решение
Geomeric Lib - это библиотека для вычисления площадей и периметров геометрических фигур.

## Описание функций

### circle.py

#### area(r)
Программа вычисляет площадь круга.
- Пример вызова: aгeа(5) -> 78.54

#### perimeter(r)
Программа вычисляет периметр круга.
- Пример вызова: aгеа(5) -> 31.42

### square.py

#### area(a)
Программа вычисляет площадь квадрата.
- Пример вызова: aгеа(4) -> 16
#### perimeter(a)
Программа вычисляет периметр квадрата.
- Пример вызова: areа(4) -› 16

### rectangle.py

#### area(a,b)
Программа вычисляет площадь прямоугольника.
- Пример вызова: агеа(5, 3) -> 15

#### perimeter (a,b)
Программа вычисляет периметр прямоугольника.
- Пример вызова: агеа (5,3) -> 16

### triangle-py

#### area(a,h)
Программа вычисляет площадь треугольника.
- Пример вызова: агеа (6,4) -> 12

#### perimeter (a,b,c)
Программа вычисляет периметр треугольника.
- Пример вызова: агеа (1,2,3) -> 6
